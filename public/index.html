Lil URL --- URL Condensing Microservice

How To Use

Input:

note: valid urls follow -- /https?:\/\/www\..+\.com/

https://lil-url.herokuapp.com/lil/https://www.mlab.com/home

Output:

{"orig_url":"https://www.mlab.com/home","lil_url":"https://lil-url.herokuapp.com/3"}

"https://lil-url.herokuapp.com/3" redirects to "https://www.mlab.com/home"
